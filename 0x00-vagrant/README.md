0x00-vagrant
This project is an introduction to using virtual machines with Vagrant. The objective of this project is to learn how to create and manage virtual machines, and to understand the basics of Vagrant.

What is Vagrant?

What will you learn in this project?
In this project, you will learn:

How to create a virtual machine with Vagrant
How to connect to the virtual machine using SSH

Project setup
This project requires you to have the following software installed:

VirtualBox
Vagrant
